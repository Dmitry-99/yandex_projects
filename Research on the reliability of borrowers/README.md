# Проект "Исследование надёжности заёмщиков"

**Описание:** на основе статистики о платёжеспособности клиентов исследовать влияет ли семейное положение и количество детей клиента на факт возврата кредита в срок.

**Статус:** выполнен.

**Цели:**
  - провести предобработку данных;
  - провести поиск дубликатов и пропусков;
  - провести категоризацию и лемматизацию;
  - сделать соответствующие выводы.
  
**Выводы:**
  - данные были обработаны: найдены дубликаты, пропуски, некорректные значения;
  - данные были категоризированы по различным признакам и условиям:
    - уровень дохода
    - количество детей
    - возраст
    - семейное положение.
  - **итоговый вывод:** было определено, что зависимость возврата кредита от семейного положения и количества детей существует.
  
**Инструменты для исследования:** предобработка данных, Python, Pandas, PyMystem3, лемматизация.

**Используемые данные:** данные о платежёспособности клиентов.
